# Biblioteca Virtual API

API REST para gestión de biblioteca con Spring Boot.

## Tecnologías
- Java 21
- Spring Boot 3.5.6
- MySQL 8.0
- Docker

## Levantar con Docker
```bash
docker-compose up -d
```

#### Acceder a: http://localhost:8080

## URLs importantes

#### Swagger: http://localhost:8080/swagger-ui/index.html
#### Health: http://localhost:8080/actuator/health
